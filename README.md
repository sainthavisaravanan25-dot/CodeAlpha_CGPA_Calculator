# CGPA Calculator

## Project Description
A C++ program that calculates semester-wise GPA and overall CGPA of a student based on grade points and credit hours.

## Features
- Student name input
- Multiple semester support
- Course-wise grade point and credit hour input
- Semester GPA calculation
- Overall CGPA calculation
- Grade classification

## Technology Used
- C++

## How to Run
1. Compile the program using a C++ compiler.
2. Run the executable file.
